# aarch64-linux-musl-gcc

Prebuilt **cross GCC toolchain** (host → `aarch64-linux-musl`, gcc 15.1.0, built with musl-cross-make) for the xlings/mcpp ecosystem.

Consumed via the `aarch64-linux-musl-gcc` xpkg in `xim-pkgindex` (XLINGS_RES sentinel).
Assets: `aarch64-linux-musl-gcc-<ver>-linux-<hostarch>.tar.gz`.
